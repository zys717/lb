# LAE-Bench: 真实案例到测试场景映射文档

# Real-to-Sim Mapping: 23 Real-World Cases → 49 Test Scenarios

## 场景扩展方法

1. **场景分解 (Scenario Decomposition)** : 将单个长案例拆解为多个独立的原子测试点（如：从“血液配送”拆解出“禁飞区识别”和“速度限制”）。
2. **参数变异 (Parameter Variation)** : 在物理仿真中引入环境扰动（如：将“常规巡检”变为“台风天气下的巡检”）。
3. **认知升级 (Cognitive Escalation)** : 注入社会性与对抗性因素（如：从“避让障碍物”升级为“多方资源公平性博弈”）。

### **一、低空物流配送 (8个案例)**

#### 案例1: 辽宁大连市探索拓展樱桃山地低空运输路径

* **核心特征** : 跨海+山地地形, 时敏+高价值货物 (生鲜), 季节性强
* **映射场景** :
* **S001 (L1)** : 单一禁飞区检测 (山地军事设施/自然保护区)
* **S006 (L1)** : 高度限制 120m (山地真高保持)
* **S007 (L1)** : 分区高度限制 (山地与平原过渡区)
* **S012 (L1)** : 时间窗口限制 (樱桃保鲜时效)
* **S027 (L2A)** : **商业压力 vs 安全裕度** (Case 1 补充) - 面对雷雨前夕的高价值樱桃运输，是否冒险起飞的决策博弈。
* **S044 (L3)** : 电池应急决策 (20km山路 + 突发电量下降)
* **S048 (L3)** : 应急疏散 (强风 8.5m/s + 低电量 15%)

#### 案例2: 江苏南京市构建低空紧急血液运输通道

* **核心特征** : 医疗急救 (LIFE_CRITICAL), 15分钟硬约束, 城市复杂空域
* **映射场景** :
* **S002 (L1)** : 多禁飞区识别 (医院周边及城市敏感区)
* **S009 (L1)** : 全局速度限制 (轻型无人机 100km/h)
* **S020 (L1)** : **审批时效** (Case 2 补充) - 血液运输对“即时审批”的极高要求（毫秒级响应）。
* **S021 (L2A)** : 电池困境 (15min 紧急任务 vs 续航焦虑)
* **S022 (L2A)** : **规则冲突优先级** (Case 2 补充) - 救护车优先权 vs 常规航路权。
* **S026 (L2A)** : 伦理三难 (救人 vs 禁飞区规则)
* **S028 (L2A)** : 动态优先级变化 (配送中途医疗等级升级)
* **S031 (L2B)** : **医疗嵌套条件** (Case 2 补充) - “如果病人情况恶化，则改飞B点”的复杂条件链。

#### 案例3: 江西赣州市发展无人机跨城低空多式联运

* **核心特征** : 跨城运输, 多式联运接驳, 长距离
* **映射场景** :
* **S014 (L1)** : BVLOS 豁免 (长距离超视距飞行)
* **S017 (L1)** : 投放物品限制 (空中转运货物交接)
* **S041 (L3)** : 机队规模权衡 (高铁联运需求预测与运力匹配)

#### 案例4: 山东青岛市开通常态化低空海岛物流航线

* **核心特征** : 跨海长距离 (10km), 恶劣天气 (海风/雾), 海岛环境
* **映射场景** :
* **S003 (L1)** : 优先级禁飞区 (海域 + 岛屿多层级限制)
* **S011 (L1)** : 夜间飞行规则 (全天候配送)
* **S019 (L1)** : **空域分类** (Case 4 补充) - 涉及军民航及海洋空域的复杂分类识别。
* **S042 (L3)** : 快充 vs 慢充策略 (单程 10min 高频次往返)

#### 案例5: 湖北十堰市打造山区库区低空物流示范

* **核心特征** : 山区库区复杂地形, 城乡配送网络
* **映射场景** :
* **S007 (L1)** : 分区高度限制 (山区 vs 平原差异化)
* **S043 (L3)** : 峰谷动态调度 (应对需求波动)

#### 案例6: 广东深圳市搭建大湾区低空物流网络

* **核心特征** : 跨城跨海, 73万架次规模化运营, 先行示范区政策
* **映射场景** :
* **S004 (L1)** : 机场管制区 (深圳宝安机场周边)
* **S010 (L1)** : **区域限速** (Case 6 补充) - 城市复杂空域，不同行政区有不同的动态限速。
* **S014 (L1)** : BVLOS 豁免 (深圳-中山 50km 跨城)
* **S023 (L2A)** : **法规更新** (Case 6 补充) - 先行示范区政策频出，测试系统对新旧法规更替的适应性。
* **S025 (L2A)** : **法规生命周期** (Case 6 补充) - 跨管辖区（不同城市）规则冲突与协调。
* **S045 (L3)** : 空域冲突解决 (高密度航线网路径优化)
* **S046 (L3)** : **垂直起降场容量** (Case 6 补充) - 高密度物流网络中，起降枢纽的排队与容量管理。

#### 案例7: 海南开展低空跨海峡运输

* **核心特征** : 超长距离跨海峡, 海况不确定性
* **映射场景** :
* **S005 (L1)** : 动态 TFR (海峡天气/演习临时禁飞)
* **S014 (L1)** : BVLOS 豁免 (海口-湛江 30km 跨海)
* **S021 (L2A)** : 电池困境 (长航时 + 恶劣海况下的应急决策)

#### 案例8: 陕西开展华山景区低空物资配送

* **核心特征** : 高海拔景区, 效率提升 (6-7h → 15min)
* **映射场景** :
* **S007 (L1)** : 分区高度限制 (高山 2000m 海拔地形豁免)
* **S017 (L1)** : 投放物品限制 (物资上山 + 垃圾下山双向管理)

### **三、低空应急救援 (3个案例)**

#### 案例21: 江苏苏州水上救援

* **核心特征** : 人机联防, 搜救范围扩大, 多方协同
* **映射场景** :
* **S015 (L1)** : 动态 NFZ 避让 (太湖水域搜救)
* **S024 (L2A)** : 信息冲突 (多源数据矛盾)
* **S028 (L2A)** : 动态优先级 (常规巡查→紧急搜救)
* **S047 (L3)** : 多方公平性 (船艇+无人机+人员协同资源分配)
* **S049 (L3)** : 应急运力调度/采购合规 (灾害场景下突发加机、资质/保险/物理能力校验)

#### 案例22: 四川公路灾情侦察

* **核心特征** : 地震应急, 灾损模型, 时序关键
* **映射场景** :
* **S016 (L1)** : 实时避让 (灾害感知)
* **S024 (L2A)** : 信息冲突 (多平台数据融合)
* **S038 (L2B)** : **时序因果** (Case 22 补充) - 地震后“先侦察、后救援”的时序逻辑，不能颠倒。
* **S039 (L2B)** : 不确定性决策 (应急勘察报告)
* **S048 (L3)** : 应急疏散 (大规模快速响应)

* **S028 (L2A)** : 动态优先级 (日常巡查→紧急险情)
* **S033 (L2B)** : **动态优先级** (Case 23 补充) - 任务性质突变导致的优先级实时跳变。

* **S049 (L3)** : 应急运力调度/采购合规 (多方协同下快速引入外部机队的资质/保险/性能审查)
